# DesignerPlatform

设计师平台 力致于工程项目的发布,接洽等

## 技术栈

Node + Express + Bootstrap + sass 


## 项目运行

```
git clone git@github.com:xmtgithub123/DesignerPlatform.git 

cd DesignerPlatform

npm install

npm start

```
# 说明

> 运行项目前,首先确保本机上已有了运行Nodejs的环境,如果没有,可以去网上查找相关教程.
